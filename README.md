## Setup
Install django
```bash
pip install django
```

Install sastrawi
``` bash
pip install Sastrawi
```

Untuk menjalankan web di local
``` bash
cd src/tubes2Algeo
```
``` bash
manage.py runserver
```
> buka http://127.0.0.1:8000/ di web browser
